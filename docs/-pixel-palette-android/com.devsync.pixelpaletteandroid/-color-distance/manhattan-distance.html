manhattanDistance

fun manhattanDistance(color1: Triple<Int, Int, Int>, color2: Triple<Int, Int, Int>): Int

Calculates the Manhattan distance (L1 distance) between two colors in the RGB color space.

The Manhattan distance measures the difference between two colors by summing the absolute differences of their red, green, and blue components. Unlike the Euclidean distance, which uses squared differences, the Manhattan distance is linear and does not consider the geometric distance but simply the cumulative difference along each axis.

The formula is as follows:

D = |ΔR| + |ΔG| + |ΔB|

Where:

  • ΔR is the difference between the red component of the two colors.

  • ΔG is the difference between the green component of the two colors.

  • ΔB is the difference between the blue component of the two colors.

This method is computationally simpler than the Euclidean distance and can be useful in scenarios where a more linear color difference is desired, or when you need a faster approximation of color differences.

Return

A Double representing the Manhattan distance between the two colors. A smaller value indicates more similarity between the colors, and a larger value indicates greater difference.

Parameters

color1

The first color to compare, represented as an object or array with RGB components.

color2

The second color to compare, represented as an object or array with RGB components.

Throws

IllegalArgumentException

If the RGB values of either color are out of valid ranges (0-255).
